Soundgrep

6-agent music production daemon — semantic sample search via CLAP embeddings, hum-to-MIDI, Ableton Live integration

Multi-agent music production system that runs as a daemon alongside Ableton Live. 6 specialist agents: sample finder (CLAP embeddings for semantic audio search), melody transcriber (hum a melody, get MIDI), arrangement assistant, mix engineer, sound designer, and session manager. Describe a sound in words, get matching samples instantly. Hum a melody, get playable MIDI.
